# 🐱 Tishka1337 Decompiler

**by InkTank1337**

---

## Что это?

Tishka1337 — продвинутый декомпилятор Java-байткода. Заточен под анализ читов для Minecraft, но работает с любыми JAR-файлами. Назван в честь моего кота Тишки. 1337 — потому что это leet speak для "elite".

---

## Возможности

- Декомпиляция на базе Vineflower — 1-в-1 как оригинал
- Автоматическая деобфускация method_XXXX → читаемые имена
- Детектор угроз: стилеры, бэкдоры, Discord вебхуки (11+ паттернов)
- Три режима работы: Ready, NoReady, Razbor
- Монохромный GUI с переливающимися 1337
- Звук успеха при завершении
- Не требует установленной Java (встроенный рантайм)

---

## Режимы

| Клавиша | Режим | Что делает |
|---------|-------|------------|
| R | Ready | Полный Gradle-проект с build.gradle, можно сразу собирать |
| N | NoReady | Только декомпилированные исходники |
| Z | Razbor | Плоская структура как у Vineflower + глубокий анализ |
